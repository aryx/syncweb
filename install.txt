You must first install a recent version of 
 - OPAM and OCaml (at least OCaml 4.12.0),
   see https://opam.ocaml.org/doc/Install.html
 - Noweb 
   see http://www.cs.tufts.edu/~nr/noweb/
 - TeX/LaTeX
   see http://www.tug.org/texlive/

Then simply type 
  $ make setup
  $ make
  $ make install

The 'syncweb' binary should then be in your PATH.
